# Banner Tracker

A lightweight banner tracking dashboard web app built with HTML, CSS, and JavaScript.

## Features
- Responsive dashboard-style UI
- Language and product metrics table
- Customizable controls and quick stats

## How to run
1. Open `index.html` in your browser.
2. Or use a local dev server (e.g. VS Code Live Server).

## Repository setup
1. Initialize git:
   ```bash
git init
   ```
2. Add files and commit:
   ```bash
git add .
git commit -m "Initial commit"
   ```
3. Create GitHub repo and push:
   ```bash
git remote add origin https://github.com/<your-username>/<repo-name>.git
git branch -M main
git push -u origin main
   ```
